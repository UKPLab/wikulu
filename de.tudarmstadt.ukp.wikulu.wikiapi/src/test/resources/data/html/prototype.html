Prototype

Open question

This open questions can also be the question in the user study.

  • How to show link names
    • The used word in the article (what you see in the article)
    • The alias name of wiki article (what you see as the link)
    • The real name of wiki article (if redirection is necessary, the link name is the article name after redirection)
  • How to show the extracted phrases
    • As normal text (not link)
    • As link (link uses wiki search)

Goal

  • User friendliness: how easy to use the user interface
  • Simplicity: keep the user interface as simple as possible
  • High learnability: how easy to learn the interface to accomplish basic task
  • High efficiency: how quickly the user to accomplish the tasks after learning
  • Lowest errors: how many errors do users make, how easy is it for users to recover from these errors
  • High memorability: after a period of non-use, how long does it take the user still remember to use it effectively
  • High satisfaction: how satisfying is the user interface (for normal users and advance users)

Guidelines

  • Offer informative feedback
  • Balance user control with automated action
  • Reduce short-term memory load
    • Support simple history mechanism
    • Integration navigation & search
  • Provide shortcuts for skilled users
  • Reduce errors; offer simple error handling
    • Avoid empty result sets
    • Address the vocabulary problem
    • Small details
  • Strive for consistency
  • Permit easy reversal of actions
  • Recognize the importance of aesthetic in design

Model of information seeking process

  • Iteration of information details (summary length)
  • Browsing as information seeking strategy (browsing VS search)
  • Integrating navigation with search
    • Categories for navigating & narrowing
    • Collapsing & extending contents
    • Sorting
  • Small history mechanism

Personalization

  • User-driven customization
    • Manual closing the snippet
    • Keeping snippet opened

To decide

  • How to show link names
    • The used word in the article (what you see in the article)
    • The alias name of wiki article (what you see as the link)
    • The real name of wiki article (if redirection is necessary, the link name is the article name after redirection)
  • How to show the extracted phrases
    • As normal text (not link)
    • As link (link uses wiki search)
